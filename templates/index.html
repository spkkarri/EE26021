About the Project

A predictive maintenance system built to forecast Remaining Useful Life (RUL) of aircraft turbofan engines using deep learning on sensor telemetry.

🎯

Objective

Estimate how many operational cycles remain before an engine requires maintenance or replacement, enabling proactive scheduling and preventing unplanned failures.

📊

Dataset

Trained on the NASA CMAPSS (Commercial Modular Aero-Propulsion System Simulation) dataset — a standard benchmark with four fault-mode sub-datasets (FD001–FD004).

🧠

Model Architecture

LSTM-based deep learning model processing multivariate time-series sensor data. Incorporates sliding window preprocessing, MinMax normalization, and dropout regularization.

🔌

API Backend

Flask REST API accepting JSON or form input, running inference in real-time. Returns predicted RUL cycles with health classification (healthy / warning / critical).

📈

Performance

Achieves RMSE ~13.4 on FD001 test set. Model evaluated against CMAPSS leaderboard benchmarks using NASA scoring function to penalize early predictions more heavily.

⚙️

Tech Stack

Python · TensorFlow/Keras · Flask · NumPy · scikit-learn. Frontend built with pure HTML5, CSS3, and vanilla JavaScript for zero-dependency embedding.
